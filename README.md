## Local preview

Open `index.html` in a browser, or:

```bash
npm run dev
```

Then visit `http://localhost:3000`.

## Deploy to Render (recommended)

### Option A — Static Site (easiest)

1. Push this folder to a GitHub repository.
2. Go to [dashboard.render.com](https://dashboard.render.com).
3. **New → Static Site**.
4. Connect the repo.
5. Settings:
   - **Build Command:** leave empty
   - **Publish Directory:** `.`
6. Create Static Site — Render will give you a URL like `https://astralforge-web.onrender.com`.

### Option B — Blueprint

1. Push to GitHub.
2. **New → Blueprint**.
3. Select the repo (uses `render.yaml`).

### Custom domain

In Render → your service → **Settings → Custom Domains**, add your domain and follow DNS instructions.
